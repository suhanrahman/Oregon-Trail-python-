# Oregon-Trail-python-
A command-line survival game inspired by The Oregon Trail, where you travel from New York to Oregon while managing resources, health, and time. Originally made in 2019.

The game simulates a 2000-mile journey where each decision matters. Players can travel, rest, hunt, and manage food while dealing with random events like sickness and obstacles. The goal is to survive the journey before winter arrives.

Features
Turn-based gameplay with user input commands (travel, rest, hunt, eat, etc.)
Resource management system (food, health, distance, time)
Randomized events (sickness, obstacles, delays)
Multiple difficulty levels (easy, medium, hard)
Dynamic date and progression system
Colored terminal output and slow text rendering for better immersion
